# نظام معرض الصور المتقدم

نظام معرض صور متطور مبني بـ React و TypeScript مع دعم كامل للغة العربية.

## المميزات

✅ **معرض صور تفاعلي** مع إمكانية التكبير والعرض التفصيلي  
✅ **نظام فلترة متقدم** حسب الفئات  
✅ **تأثيرات حركية جميلة** باستخدام Framer Motion  
✅ **تصميم متجاوب** لجميع الأجهزة  
✅ **دعم كامل للغة العربية** مع اتجاه RTL  
✅ **سهولة التخصيص والتركيب**
✅ **تحميل مسبق ذكي للصور** مع تخزين مؤقت متقدم
✅ **تحسين أداء التحميل** مع مؤشرات تقدم
✅ **دعم تنسيقات الصور الحديثة** (WebP, AVIF)
✅ **تحميل تدريجي** مع placeholders ذكية

## التركيب

1. انسخ مجلد `gallery-system` إلى مشروعك
2. تأكد من وجود المكتبات المطلوبة:

```bash
npm install framer-motion lucide-react
# أو
yarn add framer-motion lucide-react
```

## الاستخدام

```typescript
import { Gallery } from './gallery-system/src';

const photos = [
  {
    id: '1',
    src: 'https://example.com/image1.jpg',
    title: 'عنوان الصورة',
    category: 'الفئة',
    description: 'وصف الصورة'
  }
];

const categories = ['الكل', 'الفئة الأولى', 'الفئة الثانية'];

function App() {
  return (
    <Gallery 
      photos={photos}
      categories={categories}
      rtl={true}
    />
  );
}
```

## تحسينات الأداء الجديدة

### التحميل المسبق الذكي
- تحميل أول 8 صور بأولوية عالية
- تحميل باقي الصور في الخلفية بشكل متوازي
- نظام تخزين مؤقت متقدم للصور المحملة

### تحسين الشبكة
- تحميل الصور على دفعات (6 صور في كل دفعة)
- تأخير قصير بين الدفعات لتجنب إرهاق الشبكة
- إعادة المحاولة التلقائية للصور الفاشلة

### تحسين التخزين المؤقت
- تخزين مؤقت ذكي مع انتهاء صلاحية (30 دقيقة)
- تنظيف تلقائي للذاكرة
- حد أقصى 100 صورة في التخزين المؤقت

### مؤشرات التقدم
- مؤشر تحميل مع نسبة الإنجاز
- عداد الصور المحملة
- حالات تحميل مرئية لكل صورة

## التخصيص

يمكنك تخصيص الألوان والتصميم عبر تمرير `className` أو تعديل ملف CSS.

## المتطلبات

- Tailwind CSS
- Framer Motion
- Lucide React

## الدعم
